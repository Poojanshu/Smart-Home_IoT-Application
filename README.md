# 🏠 Smart Home Automation System using Cisco Packet Tracer IoT 

## 📌 Project Description

This project demonstrates a **Smart Home Automation System** designed and simulated using **Cisco Packet Tracer IoT environment**.

The system uses a **Home Gateway as a central controller** to connect and manage multiple IoT devices wirelessly. Users can monitor and control home appliances such as lights, fans, doors, and sensors using a laptop or smartphone through the IoT network.

The project shows how IoT technology can be used to improve **comfort, security, and energy management** in a modern home.

## 🔄 Working Principle

1. All IoT devices are connected to the Home Gateway using a wireless network.
2. The Home Gateway manages communication between devices and users.
3. Laptop and smartphone connect to the same WiFi network.
4. Users access the IoT interface through a web browser.
5. Appliances can be monitored and controlled remotely.
